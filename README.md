WebSec Copilot 🚀

AI-Powered Security Copilot for Modern “Vibe-Coded” Web Apps

> A full-stack security auditing platform that scans websites, repositories, packages, CVEs, and code — powered by AI and real-world threat intelligence.




---

🔥 Why This Project

In today’s AI-driven development era, developers are building faster than ever.

But there is a serious problem.

Security is being ignored.

From exposed APIs to leaked data and vulnerable deployments, the number of cyber attacks is rapidly increasing — especially due to insecure development practices.

WebSec Copilot is built to solve this.

It makes security simple, fast, and actionable — so developers can build fast without compromising safety.


---

🔍 What WebSec Copilot Scans

🌐 Websites → headers, TLS, cookies, CSP, exposed endpoints

🧠 JavaScript → secrets, tokens, unsafe patterns

📦 Packages → known vulnerabilities & risk analysis

🐙 GitHub Repositories → code issues & misconfigurations

🧾 Code Snippets → AI-based vulnerability detection

🚨 CVEs → real-world exploit intelligence



---

⚡ Features

🔍 Multi-source audit (URL, repo, CVE, package, snippet)

🛡️ Runtime security scanning

💻 Code vulnerability detection

🧠 AI-powered explanations & fixes

📡 Real-world threat intelligence

🔧 Remediation board (prioritized fixes with code solutions)

📊 Security scoring system

📄 Exportable reports (PDF / JSON / Markdown)

🎮 Demo mode for judges

⚙️ Local Setup

git clone https://github.com/CodeWithMehru/WebSec-Copilot.git
cd websec-copilot
npm install

Setup Convex

npx convex dev

Run App

npm run dev


---

🔐 Environment Variables

Create a .env.local file and add:

NEXT_PUBLIC_CONVEX_URL=your_convex_url
EXA_API_KEY=your_exa_key
APIFY_API_TOKEN=your_apify_token
GITHUB_TOKEN=your_github_token
GROQ_API_KEY=your_groq_key
CONVEX_DEPLOYMENT=your_convex_deployment
NEXT_PUBLIC_CONVEX_SITE_URL=your_convex_site_url
